# lowply.github.io

[lowply.github.io](https://lowply.github.io) - hosted by [GitHub Pages](https://pages.github.com/)

### Build

```bash
$ make build
```

### Run dev server

```bash
$ make local
```

### Write a new blog post

```bash
$ ./script/new slug
```

### Sync assets

```bash
$ ./script/push-assets
```
